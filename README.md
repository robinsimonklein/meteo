# meteo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
### Warning Important

Please use HTTPS for GeoLocalisation. Browsers doesn't allow Geoloc on HTTP.
